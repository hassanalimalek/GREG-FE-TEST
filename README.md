# AI Frontend Test Results

This repository contains the results of testing three major AI assistants (Claude, ChatGPT, and Gemini) on a frontend development task.

## Structure

- **chatgpt/** - ChatGPT's attempts at the frontend test
  - v1-initialprompt.html - Initial attempt
  - v2-iteration.html - Second iteration
  - v3-final.html - Final version

- **claude/** - Claude's attempts at the frontend test
  - v1-intialprompt.html - Initial attempt
  - v2-iteration.html - Second iteration
  - v3-final.html - Final version

- **gemini/** - Gemini's attempts at the frontend test
  - v1-initialprompt.html - Initial attempt
  - v2-iteration.html - Second iteration
  - v3-final.html - Final version

- **docs/prompt.md** - The prompt used for testing

- **v4-FINAL.html** - Final optimized version

## Purpose

This test compares how different AI assistants handle frontend development tasks, tracking their iterations and improvements.
